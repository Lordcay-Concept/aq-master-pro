# A&Q Master Pro - Queue Management System

A professional queue management system for businesses to manage customers, appointments, and service queues efficiently.

## Features

- **Customer Portal**: Self-service kiosk, ticket generation, appointment history, payment history, digital QR tickets
- **Staff Dashboard**: Queue management, call customers, complete services, performance tracking
- **Admin Panel**: User management, service configuration, appointment oversight, payment analytics, system settings
- **Payment Integration**: Paystack payment processing
- **Multi-language Support**: English, Yorùbá, Igbo, Hausa, French
- **Real-time Notifications**: SMS, Email, WhatsApp alerts

## Tech Stack

- Next.js 16.1 (App Router)
- TypeScript
- TailwindCSS
- MongoDB Atlas
- NextAuth.js
- Paystack
